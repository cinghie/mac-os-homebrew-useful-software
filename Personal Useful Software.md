# Mac OS Homebrew Personal Useful Software
Use Homebrew to install all Personal Useful Software for your Mac OS

## Install Homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Update Homebrew

```
brew update
brew upgrade
```
